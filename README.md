# 🧠 Code Exploration Challenge – Task Manager CLI

## 📌 Overview
This project is part of the AI Code Exercises.  
The goal was to explore and understand an existing codebase using AI tools.

I worked with a Task Manager CLI application built with Node.js, where I learned how different parts of the system connect and work together.

---

## ⚙️ Features
- Create tasks with title, description, priority, due date and tags
- List all tasks with filters (status, priority, overdue)
- Update task status and priority
- Add and remove tags
- Delete tasks
- View statistics of tasks

---
